COMP 3104: DevOps - An Overview

COMP 3104 focuses on the practices, tools, and methodologies that combine software development (Dev) and IT operations (Ops). The course introduces students to key DevOps concepts such as continuous integration, continuous delivery, and automation, which are critical in today's fast-paced software development environments.

Key Topics Covered:

Version Control Systems: Students learn how to use Git for managing and tracking code changes across multiple team members, implementing best practices in collaborative development.
Continuous Integration (CI): The course highlights the importance of automating the integration of code changes into a shared repository, ensuring that the system is always in a working state.
Continuous Delivery (CD): COMP 3104 covers how to automate the deployment of applications, enabling teams to release software updates frequently and reliably.
Containerization: By using Docker, students explore how to package applications and their dependencies into containers, ensuring consistency across development and production environments.
Infrastructure as Code (IaC): This topic introduces automating infrastructure setup and management using tools like Terraform and Ansible.
